## MyGardenOS Project Rules

- This project is a mower web-control exploration project.
- Web commands should be implemented in small steps.
- Before modifying code, check whether this is a git repository.
- If it is a git repository, run `git status` first and avoid overwriting existing user changes.
- If it is not a git repository, create backups of modified files under `.codex-backups/YYYYMMDD-HHMMSS/`.
- Do not modify more than 3 files without explaining why.
- After editing, summarize changed files and how to roll back.
